# 충남대학교 2026학년도 입시 점수 계산기 (비공식)

이 프로젝트는 **2026학년도 충남대학교 대학입학전형 기본계획**을 바탕으로 수험생이 자신의 성적을 입력하여 환산 점수를 미리 계산해볼 수 있도록 돕는 웹 애플리케이션입니다.

> **⚠️ 주의사항**: 본 서비스는 학교 공식 프로그램이 아닌 **비공식** 툴입니다. 실제 합격 여부 및 정확한 산출식은 충남대학교 입학처의 모집요강을 반드시 확인하시기 바랍니다.

## 주요 기능

### 1. 수시 (학생부교과) 점수 환산
- **학생부교과 100% 전형** 기준 계산
- **일반선택과목**: 석차등급(1~9등급) 및 이수단위 반영
- **진로선택과목**: 성취도(A/B/C) 및 성취도별 분포 비율에 따른 변환 등급 자동 산출 (충남대 산출식 적용)
- **감점 처리**: 학교폭력 조치사항에 따른 감점 로직 적용
- **제외 학과 안내**: 면접/실기가 포함된 사범대 일부 및 예술대학 제외 알림

### 2. 정시 (수능 위주) 점수 환산
- **수능 100% 전형** 기준 표준점수 환산
- **모집단위별 반영 비율 자동 적용**:
  - 인문계열: 국45 / 수25 / 탐30
  - 자연계열(일반): 국25 / 수45 / 탐30 (과탐 가산점 로직 포함)
  - 자연계열(의약학): 수학 미적/기하, 과탐 2과목 필수 체크
  - 예체능 및 자율전공학부 별도 비율 적용
- **가산점 자동 계산**: 자연계열 지원 시 과학탐구 응시자에 대한 표준점수 10% 가산
- **감점 처리**:
  - 영어/한국사 등급별 차등 감점
  - 학교폭력 조치사항 감점

## 🛠 기술 스택 (Tech Stack)

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Design**: Responsive Web Design (Mobile/Desktop 지원), CSS Variables 활용
